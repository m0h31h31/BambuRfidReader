# 可直接使用Android手机识别拓竹RFID标签并带有库存管理的应用程序


数据库与耗材信息配置文件存在Android/data/com.m0h31h31.bamburfidreader/files中,注意备份
耗材信息配置文件来源于拓竹官方Bambu Studio 程序目录 Bambu Studio\resources\profiles\BBL\filament\filaments_color_codes.json 或者 C:\Users\<username>\AppData\Roaming\BambuStudio\system\BBL\filament\filaments_color_codes.json

https://makerworld.com.cn/zh/models/2020787  发布了一个logo模型,欢迎助力和打印,以表支持(可以缩小打印,以减少耗材浪费和打印时间)
